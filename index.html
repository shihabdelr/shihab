Solo Project 2

Solo Project 2 is a cloud-based collection manager that lets users browse, add, edit, and organize records through a responsive web interface. It stands out because it takes a simple CRUD concept and turns it into a full deployed application with a real client-server workflow. The project uses a Flask backend hosted on Render with JSON persistence, separating the frontend experience from the server-side data layer.
